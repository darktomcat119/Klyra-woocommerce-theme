TECHNICAL NOTES
===============

- All images use object-fit:cover CSS property
- Images are set to width:100%; height:100%
- This ensures images fill entire container space
- Larger images = better coverage, no empty space
- Images are cropped from center

OPTIMIZATION
============

- Compress all images before use
- Use WebP format when possible
- Target file sizes as specified above
- Use tools: TinyPNG, ImageOptim, Squoosh
- Balance quality vs file size
